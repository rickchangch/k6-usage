# k6-usage

To record the usage of K6 (an open-source load testing tool).

## Install packages

- [official link](https://k6.io/docs/getting-started/installation/)

## Getting start

- modify `script.js`
- then execute
```bash=
bash run.sh -u 2 -d 2s

# or
k6 run -u 2 -d 2s script.js
```
